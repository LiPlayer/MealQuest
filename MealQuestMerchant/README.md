# MealQuestMerchant

Expo merchant cockpit app.

## Setup

1. Copy env file and set backend URL:

```powershell
Copy-Item .env.example .env
```

2. Install deps:

```powershell
npm install
```

## Run (Android dev client)

```powershell
npm run dev:android
```

## Run (iOS dev client)

```powershell
npm run dev:ios
```

## Lint / Typecheck

```powershell
npm run lint
npm run typecheck
```
